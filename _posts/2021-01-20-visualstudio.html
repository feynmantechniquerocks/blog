You need an IDE (Integrated Development Environment) to write programs. An IDE consists of some tools that would help you as you write your own program.


Before writing our first “Hello World” program, we will download and install an IDE.


When it comes to C#, Visual Studio is the best choice.


In this post, I’m going to talk about how to download and install Visual Studio. I’d also mention the differences between different editions.


1. Visual Studio Editions

There are 3 Visual Studio Editions.

  • Community
  • Professional
  • Enterprise

Editions are self explanatory and I don’t want to talk about them. If you want to compare the editions please visit this page.


The important edition for us (wanna be C# developers) is Community Edition. We are going to use it to learn C# and develop applications. It’s completely free.


2. How to Download Visual Studio Community Edition ?

It’s so easy to download Visual Studio Community Edition. Just go ahead and click here. This link will automatically download the vs_community.exe file which is the Visual Studio Installer software.


If you want to visit the download page of the Visual Studio. Visit this page.


3. How to Install Visual Studio Community Edition ?

Start the Visual Studio Installer (The exe file you’ve just downloaded). Follow the instructions and it will take you to a page that looks like below.



You need to select the tools you’d need while developing applications. This could seem scary for the beginners but you really don’t have to worry about choices here. All you have to do is to select .NET desktop development. If you need other tools, you can always run Visual Studio Installer then you can download and install the tool you’d need.


Most recent version of Community Edition is 2019 but Visual Studio Installer allows you to download and install 2017 version as well.


I suggest you to go with version 2019 but of course it’s up to you.


We have our IDE installed. It is up and ready. Let's start programming.
