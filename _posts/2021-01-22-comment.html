This is going to be a short post.


In the last post, we've written our Hello World program. This program is simple and basically does nothing. We are going to write more complex programs in the future. We need to leave some reminders for us and for other people who might read the source code later on.


These reminders are called comments in the programming world. Comments are too important and we need to learn how to use them effeciently.


There are three ways to make comments in C#.


The first one is two slashes: //


{% highlight c# %} // This is the first comment. // You can make as many comments as you'd like // You can make multiline comments as well. Console.WriteLine("Comments are so fun !!!"); // Comments also can be here. {% endhighlight %}

The second one is the slash and the asterisk: /* */


{% highlight c# %} /* This is the first comment. You can make as many comments as you'd like You can make multiline comments as well. */ Console.WriteLine("Comments are so fun !!!"); /* Comments also can be here. */ {% endhighlight %}

The third one is called XML Documentation Comments. We'll learn about this one when we talk about methods. For now, let's skip it.
